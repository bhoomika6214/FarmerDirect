# FarmerDirect

FarmerDirect is a mobile-friendly web application that connects farmers directly with consumers and retailers.  
It enables fair pricing, secure transactions, and market access without middlemen.

## Features (Planned)
- Farmers can list their produce online.
- Direct price negotiation between farmers and buyers.
- Secure payment gateway for transactions.
- Local language support for accessibility.
- Delivery and logistics integration.

## Tech Stack (Planned)
- **Frontend:** HTML, CSS, JavaScript, React.js
- **Backend:** Node.js, Express.js
- **Database:** SQL (MySQL / PostgreSQL)
- **Hosting:** GitHub Pages (frontend), Render/Vercel (backend)
